실제 기상 시간 (Wake-up Detection):

방법 1 (스마트폰 센서): 가속도계/자이로스코프 데이터를 통해 "휴대폰이 들려졌는지" 혹은 "화면이 켜졌는지"를 감지합니다.

방법 2 (알람 Dismiss): Kkokki 앱의 알람이 '해제(Dismiss)'되는 시점을 기상 시간으로 간주합니다.

실제 출발 시간 (Departure Detection):

방법 1 (GPS Geofencing): 사용자의 '집' 좌표를 기준으로 반경 50m~100m를 벗어나는 시점을 감지합니다.

방법 2 (네트워크 전환): 집 Wi-Fi 연결이 끊기고 LTE/5G로 전환되는 시점을 보조 지표로 활용합니다.

해커톤 데모 팁: 실제 GPS 이동을 보여주기 어려우므로, 코드 내에 simulate_leaving_home() 함수를 만들어 버튼 클릭 시 "집을 나갔음"이 감지된 것처럼 시나리오를 짜는 것이 효과적입니다.
